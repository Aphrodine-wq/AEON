Changelog

Release history for AEON. See what's new, what's fixed, and what's changed.

February 22, 2026
v1.1.0

New AEON 1.1 — 30+ Engines, 5 New Detection Categories

  • New API Contract Verification — missing input validation, unchecked returns, use-after-close
  • New Numeric Safety Analysis — integer overflow/underflow, lossy coercions, float == pitfalls
  • New Null/Undefined Safety — null dereference, missing null checks, unsafe optional unwrap
  • New Error Handling Verification — swallowed exceptions, missing error propagation, cleanup gaps
  • New Dead Code Detection — unreachable code, unused variables/params, duplicate branches
  • Improved Symbolic Execution: array bounds checking, format string detection, integer overflow
  • Improved Taint Analysis: log injection, header injection, template injection, open redirect
  • Improved Concurrency: thread-unsafe collection detection, missing atomic/volatile
  • Improved Separation Logic: buffer overflow, uninitialized memory reads, unsafe buffer ops
  • Improved Website redesigned for clarity — plain-English explanations throughout
February 20, 2026
v1.0.0

New AEON 1.0 — Production Release

  • All 25 verification engines stable and production-ready
  • 14 language adapters (Python, Java, JS, TS, Go, Rust, C, C++, Ruby, Swift, Kotlin, PHP, Scala, Dart)
  • VS Code extension with inline diagnostics, CodeLens, and quick fixes
  • SARIF 2.1.0 output for GitHub Code Scanning integration
  • Baseline / diff mode for incremental adoption
  • Parallel scanning with configurable worker count
  • REST API server for programmatic verification
  • Comprehensive documentation site with tutorials and examples
  • Docker image, Homebrew tap, npm wrapper, Scoop bucket
February 15, 2026
v0.9.0

New Improved Release Candidate

  • New Bounded Model Checking engine (Clarke et al. 1986)
  • New Shape Analysis engine (Sagiv et al. 2002)
  • Improved Symbolic execution now uses KLEE-style path prioritization
  • Improved Abstract interpretation: added delayed widening (3 iterations)
  • Fixed False positive in taint analysis for parameterized queries
  • Fixed Z3 timeout handling now properly reports AEON-W008
February 8, 2026
v0.8.0

New Multi-Language Expansion

  • New Added 8 new language adapters: Ruby, Swift, Kotlin, PHP, Scala, Dart, C, C++
  • New Universal contract syntax via comment annotations (@requires, @ensures)
  • New Auto-detection of language from file extension
  • Improved Java adapter now uses javalang for full AST parsing
  • Fixed JavaScript arrow function parsing in regex adapter
