When to Use Each Tool

ESLint + AEON

ESLint for code style & linting. AEON for mathematical correctness proofs. They complement perfectly.

mypy + AEON

mypy for type annotations. AEON for runtime behavior verification. Your type hints help AEON produce better results.

Semgrep + AEON

Semgrep for pattern rules. AEON for proof-based verification. Same SARIF pipeline, different strengths.

AEON.

Formal verification for the languages you already use. 30+ engines. 14 languages.
